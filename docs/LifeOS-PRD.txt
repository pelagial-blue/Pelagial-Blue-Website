3.2 Detailed Feature List
	•	Quick‑Add Triage Box – multiline textbox; each line becomes a task on Enter. Inline spinner for severity (default 3). Shortcut Ctrl+Alt+T focuses the box.
	•	Agenda Panel – events grouped “Today”, “Tomorrow”, “Later This Week”. Color‑code work vs personal calendars (configurable). Hover tooltip shows location, attendees, notes.
	•	AI Query Panel – single‑line input with autocomplete for common commands (today, free, prioritize). Response area supports markdown; copy‑to‑clipboard button.
	•	Prioritisation Logic (fallback if AI disabled) – highest severity + earliest deadline. Displays “Suggested Task” badge next to top candidate.
	•	Local Persistence – SQLite DB (lifeos.db) in user’s app‑data folder. Automatic backup on each app close.
	•	Security & Privacy – AES‑256 encryption at rest (user‑provided passphrase). Calendar tokens stored in OS keyring. No telemetry unless opted‑in.
	•	Accessibility – high‑contrast theme toggle; full keyboard navigation.

6. Technical Architecture
	•	Frontend – Electron (JS/HTML/CSS) or Tauri (Rust + WebView) for native cross‑platform UI.
	•	Backend / Data Layer – Node.js (Electron) or Rust (Tauri) handling SQLite, encryption, OAuth flows.
	•	AI Engine – Default: Local GGML‑quantized LLM (Llama‑2‑7B) via llama.cpp. Optional Remote: OpenAI/Anthropic API behind user‑provided key (toggle).
	•	Calendar Connectors – Libraries such as node-google-calendar, node-outlook-calendar, or generic ical.js.
	•	Encryption – libsodium or built‑in crypto module (AES‑256, PBKDF2‑derived key).
	•	Packaging – electron-builder or cargo-tauri to produce installers for Windows, macOS, Linux.

9. Acceptance Criteria (MVP)
	•	Installable binary runs on Windows/macOS/Linux.
	•	Single window opens with three panels (agenda, tasks, AI).
	•	User can add/edit/delete tasks with severity scores in ≤ 2 seconds.
	•	Calendar events from at least one provider appear correctly.
	•	AI query “What does my day look like?” returns a readable summary within 2 seconds.
	•	All data persisted locally and encrypted; on restart, previous state restored.
	•	No scrolling required to see the three core panels on a 1080p monitor.
